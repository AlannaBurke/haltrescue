# Essential Supplies for Pet Degus

Degus are intelligent, social, and highly active rodents that require a specific environment to thrive. Their powerful chewing instincts and unique dietary needs mean that setting up their home correctly is crucial. This checklist covers the essential supplies you'll need to create the perfect degu habitat.

## The Degu Shopping Checklist

| Category | Item | Key Considerations |
| :--- | :--- | :--- |
| **Housing** | **Cage** | A large, multi-level, all-metal cage is mandatory. Degus will chew through any plastic components. A cage like the Critter Nation (with plastic pans replaced) is a popular choice. The more space, the better. [1] |
| | **Bedding/Substrate** | A paper-based bedding like Carefresh or kiln-dried aspen shavings are safe. Provide a deep layer for digging, or a dedicated "dig box" within the cage. Avoid cedar and pine shavings. [2] |
| **Food & Water** | **Hay** | Unlimited, high-quality grass hay (like Timothy or Orchard grass) is the most important part of their diet and should be available 24/7. [3] |
| | **Pellets** | A high-quality pellet formulated specifically for degus or chinchillas (with no sugary treats). Brands like Oxbow or Science Selective are good choices. |
| | **Water Bottle** | A glass water bottle is essential, as degus will destroy plastic ones. |
| | **Food Bowl** | A heavy ceramic bowl that can be attached to the cage is ideal to prevent tipping. |
| **Enrichment** | **Exercise Wheel** | A large, solid-metal wheel (12 inches or larger) is a necessity. Avoid plastic wheels and any wheels with a mesh running surface. [4] |
| | **Sand Bath** | A dust bath with chinchilla sand (not dust) is crucial for keeping their coats clean and healthy. Provide it for about 15-20 minutes several times a week. |
| | **Chew Toys** | This is vital. Provide a constant supply of safe woods like apple, willow, or pear branches, as well as wooden blocks and toys. Cardboard is also great for shredding. |
| | **Ledges & Platforms** | Wooden ledges and platforms should be used to create multiple levels and climbing opportunities within the cage. |
| | **Hiding Places** | Multiple hideouts made of wood or other chew-safe materials should be provided to give them a sense of security. |
| **Health & Safety** | **Travel Carrier** | A small, secure, chew-proof carrier for vet visits. |
